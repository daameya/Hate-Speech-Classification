## How to run?

Clone the repository
```bash
git clone https://github.com/daameya/Hate-Speech-Classification.git
```

```bash
conda create -n hate python=3.8 -y
```

```bash
conda activate hate
```

```bash
pip install -r requirements.txt
```

```bash
python app.py
```
